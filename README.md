### Nyaa Mirror
一个简单易用的<a href="https://nyaa.si">Nyaa</a>镜像站
（作为本人学习React三个月的练手）

### 环境要求
- NodeJS >= 20
- Python >= 3.10.0
- 部署端可以访问到Nyaa.si

> 没啥具体要求，只要能跑起来就行 = =

### 依赖安装
```bash
yarn install && pip install -r requirements.txt
```

### 运行
- 代理服务
```bash
yarn run server
```

- WebAPP
```bash
yarn run dev
```
